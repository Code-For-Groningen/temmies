A python library which interacts with [Themis](https://themis.housing.rug.nl/). Uses bs4. I'll try to end development on a somewhat working state. 

## Intended Features
* [x] Log in  
* [x] Submit
* [x] Bulk download of test cases and files
* [x] Submission status
